# Expense Tracker Android Application

A feature-rich Android application developed to help users track, analyze, and manage their personal expenses efficiently. The app allows users to record daily transactions, set budgets, visualize spending patterns, and export data — all while working completely offline.

---

## 📱 Features

- User authentication with Login & Registration
- Secure PIN-based access with Forgot Username/Password
- Add, edit, and delete expenses with categories and notes
- Daily / Weekly / Monthly budget setup
- Budget limit alerts when spending exceeds limits
- Category-wise expense visualization using Pie Charts
- Export expense data to CSV format
- Offline functionality using local SQLite database
- Clean, user-friendly UI built with Material Design

---

## 🛠️ Tech Stack

- **Language:** Kotlin  
- **UI Design:** XML (Material Design Components)  
- **Database:** SQLite (Local Storage)  
- **IDE:** Android Studio (Giraffe or newer)  
- **Build Tool:** Gradle  

---

## 🧩 Application Modules

- **User Authentication Module**
  - Login, Registration, Forgot Username & Password
- **Expense Management Module**
  - Add, edit, delete expenses with validation
- **Budget Management Module**
  - Set budget limits and receive alerts
- **Expense Analysis Module**
  - Category-wise summaries and charts
- **Export & Settings Module**
  - CSV export and profile management
